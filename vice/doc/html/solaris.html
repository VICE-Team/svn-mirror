Download VICE for Solaris / SunOS systems

You can download the latest VICE distributions for Solaris systems here. Some distributions may not be available at the release date.

Source distributions

  • Tarball: vice-2.3.tar.gz

Binary distributions

  • Binary for M68K SunOS systems: SDLVICE-2.3-m68k-sunos.tgz.
  • Binary for Sparc SunOS systems: SDLVICE-2.3-sparc-sunos.tgz.
  • Binary for Intel Solaris 2.6 systems: vice-2.3-sol26-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 7 systems: vice-2.3-sol7-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 8 systems: vice-2.3-sol8-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 9 systems: vice-2.3-sol9-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 10 systems: vice-2.3-sol10-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Intel Solaris Express/OpenSolaris/Solaris 11 systems: vice-2.3-sol11-x86-local.gz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 2.6 systems: vice-2.3-sol26-sparc-local.gz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 7 systems: vice-2.3-sol7-sparc-local.gz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 8 systems: vice-2.3-sol8-sparc-local.gz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 9 systems: vice-2.3-sol9-sparc-local.gz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 10 systems: vice-2.3-sol10-sparc-local.gz. To be installed with the Solaris installer.

Additional information for the Solaris packages can be found at the Sun Freeware homepage.

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
