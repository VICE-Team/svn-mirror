Download VICE for Solaris / SunOS systems

You can download the latest VICE distributions for Solaris and SunOS systems here. Some distributions may not be available at the release date.

Source distributions

  • Tarball: vice-2.4.tar.gz

Binary distributions

  • Binary for M68K SunOS systems: SDLVICE-2.4-m68k-sunos.tgz. Unpack in /usr/local
  • Binary for Sparc SunOS systems: SDLVICE-2.4-sparc-sunos.tgz. Unpack in /usr/local
  • Binary for Intel Solaris 2.5 systems: vice-xaw-2.4-sol25-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 2.6 systems: vice-xaw-2.4-sol26-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 7 systems: vice-xaw-2.4-sol7-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 8 systems: vice-xaw-2.4-sol8-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 9 systems: vice-xaw-2.4-sol9-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 10 systems: vice-xaw-2.4-sol10-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris 11 systems: vice-xaw-2.4-sol11-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Intel Solaris Express/OpenSolaris systems: vice-xaw-2.4-osol-x86-local.pkg.tgz. To be installed with the Solaris installer.
  • Binary for Sparc Solaris 2.5 systems: vice-xaw-2.4-sol25-sparc-local.pkg.tgz. To be installed with the Solaris installer.

Additional information for the Solaris packages can be found at the Sun Freeware homepage.

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
