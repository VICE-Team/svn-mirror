Download VICE for SCO based systems

You can download the latest VICE distributions for SCO based systems here. Some distributions may not be available at the release date.

Source distributions

  • Tarball: vice-2.4.tar.gz

Binary distributions

  • Binary for Openserver 5.x systems: vice-xaw-2.4-osr5-x86-local.pkg.tgz. To be installed with the package installer.
  • Binary for Openserver 6.x systems: vice-xaw-2.4-osr6-x86-local.pkg.tgz. To be installed with the package installer.
  • Binary for Unixware 7.x systems: vice-xaw-2.4-uw7-x86-local.pkg.tgz. To be installed with the package installer.

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
